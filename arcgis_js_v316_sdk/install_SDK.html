Install documentation (SDK) for the ArcGIS API for JavaScript

The ArcGIS JavaScript API SDK can be copied to your Web server directory. This SDK is equivalent to the version found on the ArcGIS for Developers JavaScript web site.

For best viewing experience, access the SDK using a web server (http/http), not the file system directly (file:)

  1. Copy \arcgis_js_api\sdk and all its contents from the zip file to your Web server. In this example the files are copied to:
    C:\Inetpub\wwwroot\arcgis_js_api\sdk
  2. The start page is "index.html".
